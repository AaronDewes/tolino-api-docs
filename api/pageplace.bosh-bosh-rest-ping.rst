===============================================
``GET http://bosh.pageplace.de/bosh/rest/ping``
===============================================

Used to check if the ebook reader is online (internet reachable).

Request
=======
GET Parameters
  ``responseStatus=204``
    Always there

Headers
  ``User-Agent``
    ``Dalvik/1.6.0 (Linux; U; Android 4.4.2; tolino Build/KOT49H)``
  ``Host``
    ``bosh.pageplace.de``
  ``Connection``
    ``Keep-Alive``
  ``Accept-Encoding``
    ``gzip``


Response
========
Status Code
  ``204 No Content``
